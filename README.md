# Rentify

Application mobile de location d'objets entre particuliers.

## Stack technique
- React Native + TypeScript
- Expo SDK 54
- Firebase (Auth, Firestore, Storage)
- React Navigation

## Fonctionnalités
- Authentification (inscription / connexion)
- Publication et gestion d'objets à louer
- Marketplace avec recherche et filtres
- Système de demande et validation de location
- Messagerie en temps réel
- Système d'avis et de notation
- Carte interactive des objets disponibles
- Assistant IA intégré
- Favoris

## Installation
```bash
npm install
cp .env.example .env
# Remplir .env avec vos credentials Firebase
npx expo start
```
