---
name: Style des messages de commit
description: Format attendu pour tous les messages de commit git
type: feedback
---

Messages courts et naturels en français.

Exemples valides :
- "label retiré sous le bouton +"
- "écran détail finalisé"
- "formulaire ajout objet en cours"

**Why:** L'utilisateur préfère un style conversationnel, pas de conventions formelles.

**How to apply:** Ne jamais utiliser de préfixes type "feat:", "fix:", "chore:" ni de phrases longues. Un groupe de mots en français, minuscules, sans ponctuation finale.
